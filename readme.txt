Source code:
Main file is the SwingMvc.java under mvc package
controllers under controller package
models under model package
views under view package
Domain and the constant file under mvc package

To Run the program:
double click the leaveapp.jar
Click Master\Staff List to show the staffs
Click Master\Add new staff to add new staff
Click Master\Delete staff to delete a staff
Click Master\Assign Supervisor to assign supervisor to a staff
Click Master\Leave List to show the leave apps
Double click the leave can approve/reject the leave app
